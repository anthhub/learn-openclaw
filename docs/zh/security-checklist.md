# OpenClaw 安全检查清单

> 部署前请逐项确认

## Skill 安全

- [ ] 所有安装的 Skill 均已通过 ClawGuard 扫描
- [ ] Skill 来源可信（官方或经过审核的第三方）
- [ ] 定期检查已安装 Skill 的更新日志

## 权限控制

- [ ] Agent 已配置最小权限原则
- [ ] 沙箱隔离已启用
- [ ] 已禁用不必要的工具访问

## 网络安全

- [ ] 启用 TLS/HTTPS
- [ ] Gateway 端口未对公网直接暴露
- [ ] 配置了速率限制

## 审计日志

- [ ] 所有 Agent 操作均有日志记录
- [ ] 日志存储满足合规保留要求
- [ ] 敏感信息已脱敏

## 合规

- [ ] 符合数据安全法要求
- [ ] 符合 PIPL 个人信息保护要求
- [ ] 已完成 CVE-2026-25253 漏洞修复
